# Project Summary
The AI Customer Service Platform is designed to enhance customer interactions through intelligent automation. It leverages AI technology to provide real-time assistance, streamline workflows, and improve customer satisfaction while integrating seamlessly with existing business systems. The platform is now publicly accessible, allowing businesses to deploy it and enhance their customer service capabilities.

# Project Module Description
The platform consists of several functional modules, including:

- **Admin Dashboard**: Provides analytics and settings management for administrators.
- **Authentication**: Handles user login and registration processes.
- **Chat Interface**: Facilitates real-time communication between customers and support agents.
- **Integration**: Offers setup options for third-party integrations like WordPress.

# Usage
To set up the project, follow these steps:
1. Install dependencies:
   ```bash
   npm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run the application:
   ```bash
   npm start
   ```
